Contexto e tutorial

Esse projeto faz parte do curso dos Programadores Cariocas da Resilia/SENAC, projeto individual módulo 2.

Não está com todos os critérios pedidos, fora de padrão, além de não ter conseguido fazer os dois funcionar na mesma página, então separei para fazer o mínimo, que é entregar o projeto com os dois funcionando. Tive menos tempo para trabalhar do que eu esperava, seja por fatores dentro do meu controle ou não. Só consegui entregar graças a colegas da minha turma que me ajudaram muito.

Tive mais dificuldade do que eu achei que teria. Futuramente planejo alterar para deixar em um padrão, além de estarem funcionando 100% dentro dos padrões originalmente exigidos. Mas antes preciso ter mais domínio no JavaScript além das funções.

Para usar e fazer funcionar é bem simples, basta escolher qual dos dois você quer utilizar, digitar o texto que você quiser, escolher se quer codificar ou decodificar.

No Cifra de César, você precisa repetir os passos anteriores e além disso clicar no botão code ou decode caso tenha escolhido decodificar.

Como disse não está padronizado, planejo alterar futuramente.
